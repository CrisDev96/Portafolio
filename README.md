# Portafolio Web Personal

Este repositorio contiene el código fuente de mi portafolio personal. El proyecto está desarrollado como una página web estática (cuyo archivo principal es `index.html`), diseñada con un enfoque minimalista y optimizada para ofrecer una experiencia fluida.

El objetivo principal del sitio es presentar mi experiencia profesional, entorno tecnológico y proyectos destacados de la forma más clara posible. He optado por una estructura en HTML para garantizar la máxima velocidad de carga rápida, priorizar la facilidad de mantenimiento y asegurar una adaptabilidad inmediata en cualquier plataforma de despliegue (como GitHub Pages o similares).
